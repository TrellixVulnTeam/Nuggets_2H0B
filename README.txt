Nuggets
=======

Version of python: <= 2.7

Install:
	Windows + Linux:
		Try these steps via the command line.
		* Be sure to have python installed.
		* run: python setup.py install
		* If setup fails install scipy via pip: pip install git+http://github.com/scipy/scipy/
		* If you do not have pip installed use: python get-pip.py
